# 👨‍💼 Vue Employee Management System

A modern Employee Management application built with **Vue.js 3** and **Vite**. This project demonstrates CRUD (Create, Read, Update, Delete) operations, component-based architecture, form validation, and REST API integration using the JSONPlaceholder API.

---

## ✨ Features

- ✅ Display Employee List
- ➕ Add New Employee
- ✏️ Edit Employee Details
- 🗑️ Delete Employee
- ✅ Form Validation
- 📡 Fetch Employee Data from API
- 📤 Send Employee Data using POST Request
- 🔄 Update Employee using PUT Request
- ❌ Delete Employee using DELETE Request
- ⚡ Responsive User Interface
- 🧩 Component-Based Architecture
- 🔥 Built with Vue.js 3 + Vite

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/ksdinesh-07/vue-employee-management.git
```

### Navigate to the Project

```bash
cd vue-employee-management
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

## 📡 API Endpoints Used

### Get Employees

```http
GET https://jsonplaceholder.typicode.com/users
```

### Add Employee

```http
POST https://jsonplaceholder.typicode.com/posts
```

### Update Employee

```http
PUT https://jsonplaceholder.typicode.com/posts/:id
```

### Delete Employee

```http
DELETE https://jsonplaceholder.typicode.com/posts/:id
```
